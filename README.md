# AtEMkart
Full-Stack AtEMkart with Admin Dashboard & Paytm Payment Gateway.

### How to run ->

- #### clone the entire repo
- #### cd AtEMkart
- #### npm install
- #### cd frontend
- #### npm install
- #### cd..
- #### npm run dev

### Node version: 16.20.0
### Python version: 3.11.0
### pip version: pip 23.1.2
